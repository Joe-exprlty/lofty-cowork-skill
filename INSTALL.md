# Installing the Lofty + Cowork Skill

Read this if someone shared the Lofty + Cowork skill with you. The full, illustrated setup walkthrough lives on the project web page. This file is a short pointer.

---

## Quick start

1. **Download the .skill file.** Go to the latest release on GitHub: [github.com/Joe-exprlty/lofty-cowork-skill/releases/latest](https://github.com/Joe-exprlty/lofty-cowork-skill/releases/latest). Click `lofty-cowork-helper.skill` in the assets list. About 42 KB.
2. **Install it.** Double-click the file you just downloaded. Claude Desktop opens with a dialog asking "Add lofty-cowork-helper to your library?" Click **Add to library**.
3. **Run setup.** Open a new Cowork conversation in Claude Desktop and type: "Set up Lofty for the first time." Claude takes it from there.

That's enough to get started. The full step-by-step walkthrough, including what to have ready (Claude Desktop, Lofty access, etc.), lives at the web page below.

## What you'll need

- A computer (Mac, Windows, or Linux)
- Claude Desktop with a Pro or Max plan for Cowork
- A Lofty account with API access enabled
- About 15 minutes

Claude handles the technical work (Python install if needed, project folder creation, file editing) so you don't need to.

---

## A note on data handling

When you ask Claude to look up, update, or summarize leads, the lead content Claude reads (names, contact info, notes, activity) is processed through Anthropic's models the same way any Claude conversation is processed. Your Lofty API key stays on your computer and is never sent to Anthropic.

Before using this tool with real client data, confirm it fits your brokerage's data handling rules, your MLS rules, and what your clients reasonably expect. This skill is provided as is. Verify behavior in your own Lofty account before relying on it for client-facing work. Not affiliated with or endorsed by Lofty Inc. or Anthropic.
