Setup

Step by step

Pick your operating system. The commands below will adjust automatically.

1

Download the skill file

Save it somewhere you can find. About 42 KB.

Download lofty-cowork-helper.skill
2

Install the skill in Claude Desktop

Open Claude Desktop. Find the Skills or Plugins panel in settings. Drag your downloaded .skill file into it, or use "Install from file."

3

Confirm it installed

Open a new Cowork conversation and ask:

Do you have a Lofty skill installed?

Claude should mention lofty-cowork-helper.

4

Pick a workspace folder

In your terminal, create the folder where Claude will install Lofty files:

mkdir -p ~/Code/lofty-tools
New-Item -ItemType Directory -Force -Path "$HOME\Code\lofty-tools"

Use PowerShell or Windows Terminal, not the older Command Prompt.

mkdir -p ~/Code/lofty-tools

In Claude Desktop, point Cowork at that folder.

5

Ask Claude to run setup

In your Cowork session, say:

Set up Lofty for the first time.

Claude walks you through getting your Lofty API key, creating the project structure, customizing your context file, and pasting your API key into .env. About 10 minutes.

Heads up: Claude will ask you to log into Lofty and generate a personal API key. Treat that key like a password. Never paste it into chat.
6

Verify it works

Try these prompts. If both return data, you are connected:

Find my most recent Lofty leads
Show me the activity for that lead

Reference

Top 5 Lofty quirks

The skill handles all of these for you. Knowing them helps when an error appears.

  1. Auth header is token, not Bearer. Mixing them up returns error 200058.
  2. /v1.0/leads ignores keyword and sort. The skill uses a leads index workaround.
  3. Activities must use v1.0. The v2.0 endpoint returns empty results.
  4. Notes go to POST /v1.0/notes with leadId in body. The intuitive path returns 404.
  5. Rate limit is 10 requests per minute. The starter client enforces 6.5 second spacing.
Safety

Habits we recommend

Built into how the skill operates. Knowing them helps you understand what Claude is doing.

  • Confirm before email or text. Always. Claude drafts and asks before sending.
  • Confirm before deleting. The Lofty UI does not have an undo.
  • Never paste your API key into chat. Treat it like a password.
  • Verify the lead ID before logging notes. A name match could be the wrong person.
  • Times use ISO 8601 with offset. Pacific is -07:00 in DST, -08:00 outside.
